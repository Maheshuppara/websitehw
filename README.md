# AÏDO Landing Page Replica

A premium, responsive HTML/CSS/JS replica of the AÏDO landing page, featuring a modern design system, interactive elements, and custom-generated visual assets.

## Features
- **Promo alert banner**: Dismissible promo banner.
- **Language Switcher**: Clickable custom dropdown menu.
- **Interactive Chat Mockup**: Real-time message swapping for Mathematics, English, and Reading subjects.
- **Mode-toggled Carousel**: Carousel showing "On their own" or "Guided session" product highlights.
- **Product specifications list**: Tabular specs list detailing dimensions, connectivity, battery, and app configurations.
- **Collapsible FAQ accordion**: Seamlessly expands answers.
- **Responsive styling**: Tailored for both desktop and mobile viewports.

## File Structure
- `index.html` - Semantic HTML structure
- `style.css` - Custom design tokens, classes, animations, and media queries
- `main.js` - JS logic controlling page interactions
- `assets/` - Generated photographic and illustrative product images

## Technologies
- Core: HTML5, CSS3, ES6 JavaScript
- Fonts: Outfit & Inter via Google Fonts
